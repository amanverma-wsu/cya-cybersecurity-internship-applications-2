# Career Development Experience: Choose Your Own Adventure 01

## Chosen Adventure: Cybersecurity Internship Applications

### About This Project

For my Career Development Experience, I chose to research and apply to cybersecurity internships for Summer 2026. As a junior pursuing a Bachelor's in Cybersecurity at Washington State University, this is a critical window for securing hands-on industry experience before my senior year. I applied to **98 positions** across a range of cybersecurity and DevOps roles, targeting companies from Fortune 500 enterprises to government agencies.

### Career Goals

**Short-term (next 1–2 years):**
- Secure a cybersecurity internship for Summer 2026 to gain hands-on industry experience in a professional security team
- Deepen practical skills in security engineering, application security, DevSecOps, and SOC operations
- Build professional relationships and mentorship connections that extend beyond the internship

**Long-term (3–5 years):**
- Transition into a full-time role as a Security Engineer, Application Security Engineer, or DevSecOps Engineer
- Continue building on my independent security research background to contribute to vulnerability disclosure and open-source security tooling
- Obtain industry certifications (CompTIA Security+, CySA+, or AWS Security Specialty) and grow into a senior technical role

### Positions Targeted

I cast a wide net across several cybersecurity sub-domains to explore where my skills and interests best align:

- **Security Engineer Intern** — Infrastructure security, cloud security, automation (e.g., Amazon)
- **Cybersecurity Engineer Intern** — Broad security operations, vulnerability analysis (e.g., PayPal, Emerson)
- **Application Security Analyst Intern** — Secure code review, vulnerability assessment, SAST/DAST (e.g., Thermo Fisher Scientific)
- **DevOps Engineering Intern** — CI/CD pipelines, IaC, cloud-native infrastructure (e.g., MasterControl)
- **Cyber Security Intern (General)** — Security monitoring, incident response, compliance (e.g., General Dynamics, Interstates)
- **SOC Analyst Intern** — SIEM analysis, threat detection, alert triage
